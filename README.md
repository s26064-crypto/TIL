# TIL
주석 (Comment)

주석은 프로그램 실행에는 영향을 주지 않고, 설명이나 메모를 남길 때 사용하는 것이다.

-한 줄 주석 (Single Line Comment)

// 로 시작한다.

// 뒤의 내용은 모두 주석 처리된다.

예시

// 백앤드 완전 개꿀잼


- 여러 줄 주석 (Multi Line Comment)

/* 로 시작

*/ 로 끝난다.

그 사이의 모든 내용이 주석이다.

예시

/*
백:백앤드는
앤:앤간한 버그는 다 찾아내는 직업계의
드:드라이버다
*/


(2) 자바(Java)란?

자바(Java)는 프로그래밍 언어이며
표준 스펙 + 구현으로 이루어져 있다.

(3) 자바 표준 스펙 (Standard Specification)

표준 스펙은 자바를 어떻게 만들어야 하는지에 대한 설계도이다.

특징

자바의 설계 문서

여러 회사가 이 문서를 기반으로 자바를 만든다

JCP(Java Community Process)에서 관리한다.

(4) 다양한 자바 구현

여러 회사가 자바 표준 스펙을 기반으로 실제 자바 프로그램을 만든다.

예시

Amazon Corretto

특징

회사마다 장단점이 다름

대부분 Windows / Mac / Linux 모두 지원

예

Amazon Corretto => AWS 환경에 최적화

(5) 자바 컴파일과 실행 과정

자바 프로그램은 2단계 과정을 거친다.

1 컴파일 (Compile)

개발자가 작성한

Hello.java

파일을 컴파일러가 변환한다.

.java => .class

예

javac Hello.java

결과

Hello.class

컴파일 과정에서

문법 오류 검사

JVM에서 빠르게 실행되도록 최적화

가 이루어진다.

2 실행

컴파일된 .class 파일을 실행한다.

예) java Hello

이때

JVM (Java Virtual Machine) 이 실행되면서 프로그램이 작동한다.

(6) 인텔리제이(IntelliJ)에서 컴파일과 실행

보통 직접 명령어를 사용해야 한다.

컴파일

javac Hello.java

실행

java Hello

하지만 IntelliJ는

컴파일과 실행을 자동으로 한 번에 처리해준다.

그래서 개발자가 명령어를 직접 입력할 필요가 없다.

컴파일된 파일은 보통

out 폴더

에 생성된다.

(1) 변수(Variable)란?
=================
값을 저장하는 이름이 붙은 공간이다.

프로그램에서 데이터를 저장하고 다시 사용하기 위해 만든다.

쉽게 말하면 값을 담는 상자라고 생각하면 된다.

예시

int age = 20;

int → 정수 타입

age → 변수 이름

20 → 저장할 값

(2) 변수 선언과 초기화


-1- 변수 선언

변수를 만들기

int age;

-정수 값을 저장할 age라는 변수 생성

-2- 변수 초기화

변수에 처음 값을 넣는 것

age = 20;
-3- 선언 + 초기화 동시에
int age = 20;

실무에서 가장 많이 사용

(3) 변수 값 변경

변수는 값이 바뀔 수 있다.

int age = 20;
age = 21;

-마지막 값은 21

(4) 변수 사용
int a = 10;
int b = 20;

int sum = a + b;

결과

sum = 30

 변수에 저장된 값을 연산에 사용 가능

(6) 변수의 타입 (자료형)

자바는 변수를 만들 때 타입을 반드시 지정해야 한다.

예시

int age = 20;        // 정수

double pi = 3.14;    // 실수

boolean flag = true; // 참/거짓

String name = "Tom"; // 문자열

=변수 타입 정리=


-byte:표현 길이가 너무 작다. 또 자바는 기본으로 4byte(int)를 효율적으로 계산하도록 설계되어 있다.


  >byte 타입을 직접 선언하고 여기에 숫자 값을 대입해서 계산하는 일은 거의 없다.


  >대신에 파일을 바이트 단위로 다루기 때문에 파일 전송, 파일 복사 등에 주로 사용한다


short:표현 길이가 너무 작다. 또 자바는 기본으로 4byte(int)를 효율적으로 계산하도록 설계되어 있다.int 를 사용 하도록 하자


float:표현 길이와 정밀도가 낮다. 실수형은 double을 사용하자


char:문자 하나를 표현하는 일은 거의 없다. 문자 하나를 표현할 때도 문자열을 사용할 수 있다. 예를 들어 string a = "a"와 같이 사용하면 된다


-타입	의미

int	정수	10

double	실수	3.14

boolean	참/거짓	true

String	문자열	"Hello"

변수 명명 규칙
-

자바에서 변수의 이름을 짓는데는 규칙과 관례가 있다.

규칙은 필수이다.규칙을 지키지 않으면 컴파일 오류가 발생한다.

관례는 필수는 아니지만 전세계 개발자가 해당 관례를 따르기 때문에 사실상 규칙이라고 생각한다

-규칙-
변수 이름은 숫자로 시작할 수 없다(예:1num, 1st)

   >그러나 숫자를 이름에 포함하는 것은 가능하다
>이름에는 공백이 들어갈 수 없다
>자바의 예약어를 변수 이름으로 사용할 수 없다(예:int, class, public)
>변수 이름에는 영문자(a-z, a-z), 숫자(0~9), 달러 기호($), 또는 밑줄(_)만 사용할 수 있다

연산자란?

+,-,*,/ 와 같이 계산을 수행하는 기호를 연산자라 한다. 자바에는 다음과 같은 다양한 연산자가 있다

연산자 종류

>산술 연산자 : +, -, *, /, %(나머지 연산자)
>증감(증가 및 감소) 연산자:++,--
>비교 연산자:==, !=, >, <, >=, <=
>논리 연산자:&&(AND), ||(OR), !(NOT)
>대입 연산자:=, +=, -=
